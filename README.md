 Introduction

In today’s data-driven world, healthcare analytics plays a vital role in understanding patient patterns and improving decision-making. However, access to real patient data is often restricted due to privacy concerns.

This project solves that problem by generating synthetic patient data that mimics real-world healthcare records. It allows users to explore, analyze, and visualize healthcare data safely while gaining hands-on experience in data analysis and visualization.
 Project Description

This project creates a dataset of 100 patients with randomly generated attributes such as age, gender, disease, sugar level, and heart rate. The dataset is then analyzed to extract meaningful insights and visualized to better understand trends and distributions.

 Features
Generates realistic synthetic healthcare data
Performs basic statistical analysis
Identifies high-risk patients
Saves dataset as a CSV file
Creates multiple data visualizations
Tech Stack
Python
Pandas
Faker
Matplotlib
Dataset Details

The dataset includes the following fields:

Patient ID – Unique identifier for each patient
Name – Randomly generated name
Age – Values between 20 and 80
Gender – Male or Female
Disease – Selected from predefined conditions
Sugar Level – Range between 80 and 250
Heart Rate – Range between 60 and 110
City – Randomly generated city
Admission Date – Random date within the current year
 Data Analysis

The project performs several analytical steps to understand the dataset:

Displays a preview of patient records
Calculates the total number of patients
Computes the average age
Counts the number of patients for each disease
Identifies patients with high sugar levels
Identifies patients with high heart rates
 Data Visualization

To make the data more interpretable, the project includes the following visualizations:

Disease Distribution (Bar Chart)
Shows how frequently each disease appears in the dataset
Gender Distribution (Pie Chart)
Displays the proportion of male and female patients
Age vs Heart Rate (Scatter Plot)
Helps identify patterns or relationships between age and heart rate
Sugar Level Distribution (Histogram)
Illustrates how sugar levels are spread across patients
